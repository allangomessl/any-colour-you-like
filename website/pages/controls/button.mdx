# Button

## Forms

```tsx live dir=row justify=space-around wrap=true
<Button>Hello Button</Button>
<Button color='red' Icon={icons.Plus} />
<Button color='orange' Icon={icons.Plus}> Hello Button</Button>
<Button color='green' Icon={icons.Plus} circle />
<Button color='blue' Icon={icons.Plus} circle placeholder='With Text'>Hello Button</Button>
```
